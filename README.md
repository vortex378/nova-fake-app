# NOVA

NOVA is a fictional SaaS platform built as a full frontend application.

It demonstrates how a real product can be designed, structured, and experienced
without relying on a backend or external services.

## What this is
- A complete SaaS-style web application
- Frontend-only
- Built with phased development and intentional UX

## Tech
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Lifecycle
NOVA was built in 8 phases:
1. Genesis
2. Awareness
3. Interpretation
4. Intent
5. Decision
6. Autonomy
7. Withdrawal
8. Artifact

## Status
Completed and frozen.

No further updates are planned.
